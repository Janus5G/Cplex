# Cplex dokumentation

Cplex er desktop-editoren til ChromaPlex/CPL.

Editoren er bygget med Rust + Slint og bruger den eksisterende Python-baserede ChromaPlex compiler/runtime via et JSON bridge-lag.

Cplex understøtter aktuelt:

- simple-CPL / legacy bytecode
- dansk CPL / CPA assembly
- CPA bundle build
- CrystalSimulator-kørsel
- full_potential_demo.cpl
